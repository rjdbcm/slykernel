Memory_Compare/memcmp [ Library ]

NAME

 memcmp

FUNCTION

Compares the first N bytes of the memory areas S1 and S2. It returns an integer less than, equal to, or greater than zero if S1 is found, respectively, to be less than, to match, or be greater than S2.

INPUTS

 S1 := First source address.
 S2 := Second source address.
 N  := Number of bytes to compare.

OUTPUTS

 int := Value determining whether there is a difference.
